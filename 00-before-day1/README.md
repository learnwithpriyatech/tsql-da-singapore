## Step 1 — Fork this repo

1. Make sure you have a GitHub account (free at github.com)
2. Click the **Fork** button at the top right of this page
3. Name your fork: `tsql-da-[your-first-name]` (example: `tsql-da-aisha`)
4. Click Create Fork

---

## Step 2 — Install GitHub Desktop

1. Download from desktop.github.com (free, no command line needed)
2. Sign in with your GitHub account
3. Clone your forked repo to your desktop: File → Clone Repository → choose yours

---

## Step 3 — Explore the folder structure

Open the folder on your desktop. You will see:
- `/00-before-day1/` — you are here
- `/day1-select-filter/` through `/day5-azure-capstone/` — one per session
- `/capstone-contoso/` — your main project
- `/references/` — cheatsheets and guides

---

## Step 4 — Confirm you can push a file

1. Open `/00-before-day1/setup-checklist.md`
2. Put a tick next to any item you have completed
3. Save the file
4. In GitHub Desktop: you will see the change listed. Add a message: "Setup progress"
5. Click **Commit to main** → **Push origin**
6. Check github.com — your change should appear online

If push works — you are ready. Send Priya a screenshot.
